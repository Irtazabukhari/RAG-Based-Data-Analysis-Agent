Microsoft OneDrive Node – Connects to cloud storage to fetch and monitor document uploads.

Default Helper (AI Agent) – The central orchestrator that manages chat memory and tool calling.

Pinecone Vector Store Node – Handles document indexing and similarity search for high-speed retrieval.

Embeddings Node (OpenAI) – Converts raw text into high-dimensional vectors for the AI to "understand."

Recursive Character Text Splitter – Chunks large files into digestible pieces to maintain context and stay within token limits.

Chat Input Node – Provides a conversational interface to ask specific questions about the stored data.

Output Parser – Formats the AI’s analysis into clean, readable responses for the user.
